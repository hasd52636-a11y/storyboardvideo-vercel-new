
## � API摄 配置工作流

### 第一步：选择配置方式

应用有两种配置方式，选择其中一种：

#### 方式 A：旧配置界面（KeySelection）
- **位置**：应用启动时的配置对话框
- **标签页 1**：🖼️ 对话及图像API
  - 支持：Gemini、神马 API、智谱 AI、OpenAI、Custom
  - 用途：配置文本生成、文生图、图生图、图像分析
- **标签页 2**：🎬 视频生成API
  - 支持：神马、大洋芋、Custom
  - 用途：配置视频生成

#### 方式 B：新配置界面（Multimedia Studio）
- **位置**：应用内 ⚙️ Configuration 标签页
- **功能**：统一配置所有 API 提供商
- **优点**：支持函数特定端点、更灵活

### 第二步：配置神马 API（推荐）

**如果使用旧配置界面：**
1. 打开应用，看到配置对话框
2. 点击 🖼️ "对话及图像API" 标签页
3. 在"Provider"下拉框中选择 **"神马 API"**
4. 输入你的神马 API Key
5. 点击"Save"保存

**如果使用新配置界面：**
1. 进入应用后，点击 ⚙️ Configuration 标签页
2. 在"Add API Provider"部分，选择 **"神马 API"**
3. 输入 API Key
4. 可选：配置函数特定端点
5. 点击"Add Provider"

### 第三步：使用功能

配置完成后，你可以使用所有功能：
- 🖼️ 文生图：Text-to-Image 标签页
- ✏️ 图生图：Image Editing 标签页
- 📝 文本生成：Text Generation 标签页
- 🔍 图像分析：Image Analysis 标签页
- 🎬 视频生成：Video Generation 标签页
- 📹 视频分析：Video Analysis 标签页

## 🐛 常见问题速查

### API 配置相关

| 问题 | 原因 | 解决 |
|------|------|------|
| 看不到神马 API 的密钥输入框 | 神马 API 在"对话及图像API"标签页中 | 点击 🖼️ "对话及图像API"标签页，在 Provider 下拉框中选择"神马 API" |
| 不知道在哪里配置 API Key | 有两个配置界面 | 使用旧界面（启动时）或新界面（⚙️ Configuration 标签页） |
| 配置后还是无法生成 | API Key 无效或配置未保存 | 检查 API Key 是否正确，重新保存配置 |
| 想用不同的 API 提供商 | 需要在配置中切换 | 在配置界面中选择不同的 Provider |

### 生成相关

| 问题 | 原因 | 解决 |
|------|------|------|
| 无法生成 | API Key 无效 | 重新配置 |
| 生成很慢 | 高峰期/复杂提示词 | 简化提示词 |
| 质量不好 | 提示词不清晰 | 改进提示词 |
| 提示词违规 | 包含真人/暴力 | 修改内容 |

### 导出相关

| 问题 | 原因 | 解决 |
|------|------|------|
| 比例错误 | 混合不同比例 | 分别导出 |
| 文件太大 | 分镜过多 | 减少数量 |
| 编号错误 | 选择顺序 | 重新选择 |

### 视频相关

| 问题 | 原因 | 解决 |
|------|------|------|
| 配额不足 | 使用过多 | 购买配额 |
| 生成失败 | 提示词违规 | 修改提示词 |
| 质量差 | 参数不优 | 启用 HD |

## 💡 最佳实践

### ✅ 做这些

- ✅ 使用具体的形容词
- ✅ 指定风格和光线
- ✅ 提供参考图片
- ✅ 简化复杂提示词
- ✅ 多次尝试选最好的
- ✅ 使用快速分镜节省时间
- ✅ 定期检查配额

### ❌ 避免这些

- ❌ 真人或类似真人的内容
- ❌ 暴力、色情等违规内容
- ❌ 版权内容或活着的名人
- ❌ 过长的提示词（>500 字）
- ❌ 模糊的描述
- ❌ 混合不同比例导出
- ❌ 高峰时段大量生成
