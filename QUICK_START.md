# 快速启动 Demo

## 一句话启动

```bash
npm run dev
```

然后打开浏览器访问：`http://localhost:5173/demo.html`

## 看到的功能

### 标签页 1️⃣ - 一键运镜 (Camera Motion Library)
**这是新功能！** 原来的"符号库"现在叫"一键运镜"

**你可以做什么：**
- ✅ 点击"+ 上传运镜" - 添加自定义运镜
- ✅ 点击"编辑" - 修改运镜名称和描述  
- ✅ 点击"删除" - 删除不需要的运镜
- ✅ 点击运镜卡片 - 生成故事板

**预设运镜：**
- ← Pan Left (左移)
- → Pan Right (右移)
- 🔍 Zoom In (放大)
- 📏 Zoom Out (缩小)
- ⭕ Orbit (环绕)
- ↑ Tilt Up (向上倾斜)

---

### 标签页 2️⃣ - 一键运动 (Action Motion)
**这是新功能！** 4个预定义的动作

**4个动作：**
- ▶️ 前进 (Forward) - 平滑前进
- 🔄 旋转 (Rotate) - 360度旋转
- ⬆️ 跳跃 (Jump) - 跳跃运动
- ✈️ 飞行 (Fly) - 飞行运动

点击任何动作生成运动序列

---

### 标签页 3️⃣ - Quick Storyboard
4个快速模板：
- 📐 Three View - 三视图
- 🔲 Multi Grid - 网格布局
- 🎨 Style Comparison - 风格对比
- 📖 Narrative - 故事进展

---

### 标签页 4️⃣ - Canvas
查看生成的图像
- 保存生成
- 清空画布

---

### 标签页 5️⃣ - History
查看生成历史
- 查看详情
- 删除记录

---

## 试试这个流程

1. **打开 Demo**
   ```bash
   npm run dev
   ```

2. **在"一键运镜"标签页**
   - 点击"+ 上传运镜"
   - 输入名称：`快速摇镜`
   - 输入描述：`快速摇动摄像机`
   - 输入图标：`🎬`
   - 点击"确定"
   - 新运镜会出现在列表中！

3. **点击任何运镜卡片**
   - 自动切换到 Canvas 标签页
   - 看到生成的 3 个视图

4. **在"一键运动"标签页**
   - 点击"前进"按钮
   - 看到生成的运动序列

5. **在 Canvas 标签页**
   - 点击"保存生成"
   - 查看 History 标签页看到记录
