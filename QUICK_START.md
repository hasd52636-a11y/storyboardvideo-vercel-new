# 快速开始指南

## 项目设置

### 1. 安装依赖
```bash
npm install
```

### 2. 启动开发服务器
```bash
npm run dev
```
服务器将在 `http://localhost:5174/` 启动

### 3. 配置API密钥

首次打开应用时，需要配置API密钥：

#### Gemini API
1. 访问 [Google AI Studio](https://aistudio.google.com)
2. 获取API密钥
3. 在应用中输入密钥

#### Sora2 API
1. 获取Sora2 API密钥
2. 在应用中配置

## 基本使用流程

### 生成分镜

1. **输入脚本或对话**
   - 在左侧边栏输入剧本文本
   - 或上传对话文件

2. **选择参数**
   - 场景数量
   - 风格（写实、动画等）
   - 画幅比例（16:9, 9:16等）
   - 时长（可选）

3. **生成图片**
   - 点击"生成"按钮
   - 等待AI生成分镜图

### 编辑分镜

1. **选择分镜**
   - 点击分镜卡片选中
   - Ctrl+A 全选

2. **编辑操作**
   - 拖拽排序
   - 右键菜单编辑
   - 添加符号标注

3. **批量操作**
   - 批量重绘
   - 批量删除

### 生成视频

1. **选择分镜**
   - 选中要生成视频的分镜

2. **配置视频参数**
   - 视频时长
   - 风格
   - 其他参数

3. **生成视频**
   - 点击"生成视频"
   - 等待处理完成

### 导出结果

1. **导出JPEG**
   - 选中分镜
   - 点击"导出JPEG"
   - 自动下载分镜板

2. **导出提示词**
   - 选中分镜
   - 点击"导出提示词"
   - 获取TXT文件

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl+A | 全选分镜 |
| Delete | 删除选中分镜 |
| Ctrl+Z | 撤销（如果支持） |

## 常见问题

### Q: 如何更改语言？
A: 点击右上角的语言按钮，选择中文或英文

### Q: 如何更改主题？
A: 点击右上角的主题按钮，切换深色/浅色模式

### Q: 生成失败怎么办？
A: 
1. 检查API密钥是否正确
2. 检查网络连接
3. 查看浏览器控制台错误信息

### Q: 如何保存项目？
A: 目前应用不支持项目保存，建议导出JPEG和提示词进行备份
