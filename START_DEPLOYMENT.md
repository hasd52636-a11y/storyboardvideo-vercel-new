# 🚀 开始部署 - Storyboard Master v2

## 立即部署

### 选择你的操作系统

#### Windows 用户
```bash
deploy-new-project.bat
```

#### macOS / Linux 用户
```bash
chmod +x deploy-new-project.sh
./deploy-new-project.sh
```

## 部署流程

1. **执行脚本** (1 分钟)
   - 清理旧文件
   - 安装依赖
   - 构建项目

2. **上传到 Vercel** (2-3 分钟)
   - 连接 Vercel 账户
   - 上传构建文件
   - 配置部署

3. **部署完成** (2-5 分钟)
   - Vercel 处理部署
   - 生成 URL
   - 项目上线

**总耗时**: 5-10 分钟

## 部署完成后

### 访问新项目
```
https://storyboard-master-v2.vercel.app
```

### 测试功能
1. 生成分镜
2. 测试批量重绘 (最多 6 张)
3. 测试分镜导出
4. 验证 UI 改进

### 配置 API 密钥
在应用中配置 Gemini API 密钥

## 常见问题

**Q: 需要 Vercel 账户吗？**
A: 是的，需要免费的 Vercel 账户

**Q: 会覆盖原项目吗？**
A: 不会，新项目是独立的

**Q: 如何更新新项目？**
A: 推送代码到 main 分支，Vercel 会自动部署

**Q: 如何删除新项目？**
A: 在 Vercel 仪表板中删除项目
