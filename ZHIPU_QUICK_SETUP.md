# 智谱 GLM 快速配置 (3 分钟)

## ✅ 配置清单

### 第 1 步：获取 API Key (1 分钟)

- [ ] 访问 https://open.bigmodel.cn/usercenter/apikeys
- [ ] 登录或注册账户
- [ ] 点击 "创建新的 API Key"
- [ ] 复制 API Key（格式：`sk-...`）

### 第 2 步：在应用中配置 (1 分钟)

- [ ] 打开应用
- [ ] 点击 **⚙️ 系统设置** → **API 接口配置**
- [ ] 选择 **智谱 GLM (推荐)**
- [ ] 粘贴 API Key
- [ ] 点击 **✨ 测试连接**

### 第 3 步：验证配置 (1 分钟)

- [ ] 看到 ✅ 连接成功提示
- [ ] 显示剩余配额信息
- [ ] 配置完成！

## 🔧 故障排除

### 问题 1：连接失败 - "API Key 无效"

**原因**：API Key 复制错误或已过期

**解决**：
1. 重新访问 https://open.bigmodel.cn/usercenter/apikeys
2. 确认 API Key 是否有效
3. 重新复制并粘贴

### 问题 2：连接失败 - "无权限访问"

**原因**：账户权限不足

**解决**：
1. 检查账户是否已激活
2. 确认是否有使用 GLM-4.6V 和 CogVideoX-3 的权限
3. 联系智谱客服

### 问题 3：视频生成失败

**原因**：可能是提示词违规或账户配额不足

**解决**：
1. 检查浏览器控制台的错误信息
2. 修改提示词，避免敏感内容
3. 检查账户配额是否充足

## 💡 使用建议

### 图片分析最佳实践

1. **使用高质量图片**
   - 分辨率至少 512x512
   - 清晰的主体和背景

2. **提供清晰的分析提示**
   - 例如："分析这张图片中的人物、场景和光线"
   - 而不是："分析这张图片"

3. **利用分析结果**
   - 用于生成更准确的视频提示词
   - 用于场景理解和内容描述

### 视频生成最佳实践

1. **选择合适的质量模式**
   - 测试阶段：使用 `speed` 模式
   - 最终输出：使用 `quality` 模式

2. **编写详细的提示词**
   - 包含主体、场景、动作、光线等信息
   - 例如："一只橙色的猫在阳光充足的花园里玩球，镜头缓慢推近"

3. **合理设置参数**
   - 分辨率：根据需要选择（1920x1080 是标准选择）
   - 帧率：30 FPS 足够大多数场景
   - 时长：5 秒用于短视频，10 秒用于长视频
