## 🚀 快速开始（3 步）

### 1️⃣ 获取 API Key
访问 [智谱 AI 官网](https://open.bigmodel.cn)，获取你的 API Key

### 2️⃣ 配置应用
- 打开应用 → 点击 **"设置"** → 选择 **"对话及图像API"**
- 选择 **"智谱 AI (ChatGLM)"**
- 粘贴 API Key
- 点击 **"测试连接"** 验证
- 点击 **"保存配置"**

### 3️⃣ 开始使用
- 在 **"创意对话"** 或 **"剧本生成"** 中使用智谱生成分镜
