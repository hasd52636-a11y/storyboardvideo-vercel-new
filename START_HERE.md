# 🚀 从这里开始

## ⚡ 30 秒快速启动

```bash
npm run dev
```

然后打开浏览器访问：**http://localhost:5173/demo.html**

---

## 👀 你会看到什么

### 标签页 1: 一键运镜 ⭐ 新功能
- 6 个预设运镜（Pan Left, Pan Right, Zoom In, Zoom Out, Orbit, Tilt Up）
- **+ 上传运镜** 按钮 - 添加自定义运镜
- **编辑** 按钮 - 修改运镜信息
- **删除** 按钮 - 删除不需要的运镜
- 点击运镜卡片生成故事板

### 标签页 2: 一键运动 ⭐ 新功能
- 4 个预定义动作：
  - ▶️ 前进 (Forward)
  - 🔄 旋转 (Rotate)
  - ⬆️ 跳跃 (Jump)
  - ✈️ 飞行 (Fly)
- 点击任何动作生成运动序列

### 标签页 3: Quick Storyboard
- 4 个快速模板
- 点击生成故事板

### 标签页 4: Canvas
- 查看生成的图像
- 保存或清空

### 标签页 5: History
- 查看生成历史
- 删除记录

---

## 🎯 立即体验

### 体验 1：创建自定义运镜（30 秒）
```
1. 点击"一键运镜"标签页
2. 点击"+ 上传运镜"
3. 输入：名称 = "快速摇镜"，描述 = "快速摇动"，图标 = "🎬"
4. 点击"确定"
5. 新运镜出现在列表中！
6. 点击运镜卡片生成故事板
```

### 体验 2：使用一键运动（20 秒）
```
1. 点击"一键运动"标签页
2. 点击"前进"按钮
3. 看到生成的运动序列
4. 点击"保存生成"
5. 在 History 标签页查看记录
```

## ❓ 常见问题

**Q: 如何上传自定义运镜？**
A: 在"一键运镜"标签页，点击"+ 上传运镜"按钮。

**Q: 一键运动有多少个动作？**
A: 有 4 个：前进、旋转、跳跃、飞行。

**Q: 生成的图像会保存吗？**
A: 点击"保存生成"会添加到历史记录。

**Q: 需要后端 API 吗？**
A: 不需要。Demo 完全独立运行。
