# 视频生成功能 - 快速参考卡片

## 🎯 三步快速开始

### 第 1 步：设置密钥（5 分钟）
```
1. 点击左侧边栏 ⚙️ 按钮
2. 输入 Base URL 和 API Key
3. 点击"测试连接"
4. 看到 ✅ 表示成功
```

### 第 2 步：上传分镜（2 分钟）
```
1. 点击左侧边栏 📥 按钮
2. 选择"上传分镜图片"
3. 选择 1-6 张图片
4. 在画布上选择要生成的分镜
```

### 第 3 步：生成视频（1-10 分钟）
```
1. 点击右侧边栏 🎬 生成视频
2. 填写视频提示词
3. 选择模型、宽高比、时长
4. 点击"生成"按钮
5. 等待完成后下载
```

## 📊 参数速查表

### 模型对比

| 模型 | 速度 | 质量 | 分辨率 | 推荐场景 |
|------|------|------|--------|---------|
| Sora 2 | ⚡⚡⚡ | ⭐⭐⭐ | 720P | 快速生成 |
| Sora 2 Pro | ⚡⚡ | ⭐⭐⭐⭐⭐ | 1080P | 高质量 |

### 宽高比

| 宽高比 | 用途 | 示例 |
|--------|------|------|
| 16:9 | 电影、广告、演示 | 1920×1080 |
| 9:16 | 短视频、手机竖屏 | 1080×1920 |

### 时长

| 时长 | 生成时间 | 用途 |
|------|---------|------|
| 10 秒 | 1-3 分钟 | 短视频、预告 |
| 15 秒 | 3-5 分钟 | 标准视频 |
| 25 秒 | 5-10 分钟 | 长视频 |

### 高清选项

| 选项 | 分辨率 | 生成时间 | 成本 |
|------|--------|---------|------|
| 标清 | 720P | 1-5 分钟 | 低 |
| 高清 | 1080P | 8-15 分钟 | 高 |

## 💡 提示词模板

### 模板 1：简单描述
```
[主体] 在 [场景] 中 [动作]，[光线/氛围]
```

**示例：**
```
一只猫在公园里奔跑，阳光洒在草地上
```

### 模板 2：详细描述
```
[主体描述]，在 [场景描述] 中 [动作描述]，
[摄像机运动]，[风格]，[光线]，[色彩]
```

**示例：**
```
一只金色的猫，在阳光洒满的公园里奔跑，
摄像机跟随猫的运动，电影级别的画质，
柔和的光线，色彩鲜艳
```

### 模板 3：场景转换
```
[场景 1] → [场景 2]，[过渡效果]
```

**示例：**
```
猫在公园里奔跑 → 猫在树下休息，平滑过渡
```

## ⚠️ 常见错误

| 错误 | 原因 | 解决 |
|------|------|------|
| 401 Unauthorized | API Key 无效 | 检查 API Key 是否正确 |
| 400 Bad Request | 参数错误 | 检查提示词和参数 |
| 429 Too Many Requests | 请求过于频繁 | 等待几分钟后重试 |
| 图片包含真人 | 参考图片有人脸 | 使用不含人脸的图片 |
| 提示词包含违规内容 | 提示词不符合规范 | 修改提示词 |

## 💾 配置备份

### 保存配置
```
1. 打开浏览器开发者工具 (F12)
2. 进入 Application → Local Storage
3. 找到 sora_base_url 和 sora_api_key
4. 复制并保存到安全的地方
```

### 恢复配置
```
1. 打开 API 配置对话框
2. 粘贴保存的 Base URL 和 API Key
3. 点击"测试连接"
```
