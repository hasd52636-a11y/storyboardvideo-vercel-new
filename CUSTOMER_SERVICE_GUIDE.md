# 分镜大师 - 智慧客服指南

## 📞 客服常见问题库

### 第一部分：API 配置问题

#### Q1: 我不知道如何获取 API Key

**A:** 根据你选择的 AI 服务商，获取方式不同：

**Gemini（推荐）：**
1. 访问 [Google AI Studio](https://ai.studio/apps/drive/1wkj1Wo9G2rTr2nQ8UPbb3VheY-hoIFap)
2. 点击 "Get API Key"
3. 复制 API Key
4. 在应用中粘贴即可

**OpenAI：**
1. 访问 [OpenAI Platform](https://platform.openai.com)
2. 登录账户
3. 进入 API Keys 页面
4. 创建新的 API Key
5. 复制并在应用中使用

**其他服务类似，访问对应官网获取 API Key。**

---

#### Q2: API Key 配置后显示"无效"

**A:** 检查以下几点：

1. **API Key 是否正确复制？**
   - 确保没有多余空格
   - 确保没有遗漏字符
   - 重新复制一遍试试

2. **API Key 是否过期？**
   - 某些服务的 API Key 有有效期
   - 访问官网检查 API Key 状态
   - 如果过期，重新生成新的

3. **网络连接是否正常？**
   - 检查网络连接
   - 尝试访问 Google 或其他网站
   - 如果无法访问，可能是网络问题

4. **是否选择了正确的服务商？**
   - 确保选择的服务商与 API Key 匹配
   - 例如：Gemini API Key 要配合 Gemini 服务商

**如果以上都检查过还是不行，请：**
- 清除浏览器缓存
- 重新启动应用
- 尝试不同的浏览器

---

#### Q3: 配置成功但无法生成分镜

**A:** 这通常是配额问题或网络问题：

1. **检查配额是否充足**
   - 点击 ⚙️ 设置
   - 查看配额信息
   - 如果配额为 0，需要购买

2. **检查网络连接**
   - 确保网络正常
   - 尝试刷新页面
   - 查看浏览器控制台错误信息

3. **尝试生成简单的分镜**
   - 使用最简单的提示词测试
   - 例如："一个红色的球"
   - 如果能生成，说明配置正常

4. **检查提示词是否违规**
   - 避免暴力、色情等内容
   - 避免真人或名人
   - 使用中文或英文，避免混用

### 第二部分：生成功能问题

#### Q4: 为什么生成速度这么慢？

**A:** 生成速度取决于多个因素：

**正常情况：**
- 简单分镜：30 秒 - 2 分钟
- 复杂分镜：2 - 5 分钟
- 视频生成：5 - 20 分钟

**加快生成速度的方法：**

1. **简化提示词**
   - 删除不必要的描述
   - 使用关键词而不是长句子
   - 例如：不要写"一个穿着蓝色衣服的人在走路"，改为"蓝衣人行走"

2. **选择更快的模型**
   - 视频：使用 Sora 2 而不是 Sora 2 Pro
   - 图片：某些模型比其他模型更快

3. **避免高峰时段**
   - 早上 8-10 点、中午 12-2 点、晚上 8-10 点 是高峰
   - 尽量在非高峰时段生成

4. **减少分镜数量**
   - 一次生成 2-4 张而不是 12 张
   - 分批生成可以更快

---

#### Q5: 生成的分镜质量不好

**A:** 提高生成质量的方法：

1. **改进提示词**
   - 更详细的描述
   - 指定风格、光线、构图
   - 例如：
     ```
     不好：一个人
     好：一个穿着黑色西装的商人，站在现代办公室里，
          阳光透过落地窗洒进来，表情严肃，电影级光影
     ```

2. **提供参考图片**
   - 上传参考主体
   - 系统会更准确地理解你的需求
   - 参考图片越清晰越好

3. **选择合适的风格**
   - 电影风格：适合故事叙述
   - 科幻风格：适合未来场景
   - 赛博朋克：适合高科技场景
   - 水墨风格：适合艺术表现

4. **使用高清模式**
   - 视频生成时启用 HD
   - 生成时间会增加，但质量更好

5. **多次尝试**
   - 同一提示词生成多次
   - 选择最满意的结果
   - 然后进行微调

---

#### Q6: 生成失败，显示"提示词违规"

**A:** 这说明提示词包含不允许的内容。常见原因：

**不允许的内容：**
- ❌ 真人或类似真人的内容
- ❌ 暴力、血腥场景
- ❌ 色情或不当内容
- ❌ 版权内容（如电影角色）
- ❌ 活着的名人

**解决方案：**

1. **移除真人内容**
   ```
   不行：一个真实的人在跑步
   可以：一个卡通人物在跑步
   可以：一个 3D 渲染的人物在跑步
   ```

2. **避免暴力内容**
   ```
   不行：一个人被打
   可以：一个人在躲避
   可以：一个人在防守
   ```

3. **使用虚构角色**
   ```
   不行：Tom Cruise 在跑步
   可以：一个穿着黑色衣服的男性角色在跑步
   ```

4. **重新表述**
   - 用更温和的词汇
   - 强调艺术性而不是真实性
   - 例如："艺术风格的人物肖像"

### 第三部分：多宫格功能问题

#### Q7: 多宫格是什么？我应该什么时候用？

**A:** 多宫格用于从多个摄像机角度展示同一物体。

**使用场景：**

1. **产品展示**
   - 展示产品的各个角度
   - 让客户全面了解产品
   - 例如：手机、椅子、汽车

2. **建筑设计**
   - 展示建筑的不同视角
   - 俯视图、侧面图、正面图
   - 帮助理解空间布局

3. **角色设计**
   - 展示角色的正面、侧面、背面
   - 用于动画或游戏制作
   - 确保角色设计的一致性

4. **场景规划**
   - 展示场景的多个视角
   - 帮助理解场景布局
   - 用于电影或游戏预制作

**关键特点：**
- ✅ 所有视图显示**同一个主体**
- ✅ 只改变摄像机角度，不改变主体
- ✅ 自动计算最优网格布局
- ✅ 自动标注摄像机视角

---

#### Q8: 多宫格生成的结果不一致

**A:** 如果生成的多个视图中主体看起来不同，可能是：

1. **提示词不够清晰**
   - 提供更详细的主体描述
   - 指定颜色、材质、特征
   - 例如：
     ```
     不好：一个苹果
     好：一个光滑的红色苹果，有绿色叶子，
          放在白色背景上，光线均匀
     ```

2. **参考图片不清晰**
   - 上传清晰的参考图片
   - 确保参考图片显示主体的主要特征
   - 避免过度遮挡或模糊

3. **帧数过多**
   - 帧数越多，一致性可能越差
   - 建议先用 4-6 帧测试
   - 如果满意再增加帧数

**改进方法：**
- 重新生成几次，选择最好的
- 使用参考图片增加一致性
- 简化提示词，只保留关键信息

### 第四部分：视频生成问题

#### Q9: 视频生成配置

**A:** 视频生成需要单独配置 API：

**配置步骤：**

1. 点击 ⚙️ 设置
2. 找到 **视频 API** 部分
3. 选择服务商：
   - **OpenAI Sora**（推荐）
   - **DYU API**（备选）
4. 输入 API Key
5. 点击 Test 验证
6. 保存配置

**获取 Sora API Key：**
- 访问 [OpenAI 官网](https://openai.com/sora)
- 申请 Sora API 访问权限
- 获取 API Key 后在应用中配置

---

#### Q10: 视频生成失败

**A:** 常见原因和解决方案：

**原因 1：配额不足**
- 症状：显示"配额不足"
- 解决：购买更多配额或等待配额重置

**原因 2：提示词违规**
- 症状：显示"审查失败"
- 解决：修改提示词，避免真人、暴力等内容

**原因 3：网络问题**
- 症状：显示"连接超时"
- 解决：检查网络连接，重试

**原因 4：API Key 无效**
- 症状：显示"认证失败"
- 解决：检查 API Key 是否正确，重新配置

**原因 5：模型不支持**
- 症状：显示"模型不可用"
- 解决：检查是否有 Sora API 访问权限

---

#### Q11: 视频生成太贵了

**A:** 视频生成成本确实比图片高。降低成本的方法：

1. **使用更短的时长**
   - 10 秒 vs 25 秒：成本差异很大
   - 先用 10 秒测试，满意后再生成 25 秒

2. **关闭 HD 模式**
   - HD 模式增加成本
   - 标准清晰度通常已足够

3. **使用 Sora 2 而不是 Pro**
   - Sora 2：成本低，速度快
   - Sora 2 Pro：成本高，质量更好

4. **批量生成**
   - 一次生成多个视频
   - 某些服务商提供批量折扣

5. **选择合适的时段**
   - 非高峰时段可能有优惠
   - 避免在高峰时段生成

---

#### Q12: 生成的视频质量不好

**A:** 改进视频质量的方法：

1. **改进提示词**
   - 更详细的场景描述
   - 指定摄像机运动
   - 指定风格和氛围
   - 例如：
     ```
     不好：一个人在走路
     好：一个穿着黑色衣服的人在城市街道上缓慢走路，
          摄像机跟随，背景是高楼大厦，夕阳西下，
          电影级光影，色彩饱和度高
     ```

2. **启用 HD 模式**
   - 生成 1080p 高清视频
   - 成本增加但质量明显提升

3. **使用 Sora 2 Pro**
   - Pro 模型质量更好
   - 支持更长的视频（25 秒）

4. **提供参考图片**
   - 上传参考图片
   - 系统会更准确地理解风格

5. **多次尝试**
   - 同一提示词生成多次
   - 选择最满意的结果

### 第五部分：导出和保存问题

#### Q13: 导出分镜时提示"分镜必须是同一比例"

**A:** 这是系统的安全检查。解决方案：

**原因：**
- 你选择的分镜使用了不同的比例
- 例如：混合了 16:9 和 9:16 的分镜

**解决方案：**

1. **只导出同一比例的分镜**
   - 先清空画布
   - 只生成 16:9 的分镜
   - 导出
   - 再生成 9:16 的分镜
   - 再导出

2. **检查分镜比例**
   - 点击分镜查看其属性
   - 确认所有分镜的比例相同

3. **分别导出**
   - 如果需要混合比例
   - 分别导出不同比例的分镜
   - 然后在其他工具中合并

---

#### Q14: 导出的 JPEG 文件太大

**A:** 减小文件大小的方法：

1. **减少分镜数量**
   - 导出 4 张而不是 12 张
   - 文件大小会显著减小

2. **降低分镜分辨率**
   - 在生成时选择较低的分辨率
   - 导出时系统会自动调整

3. **使用图片压缩工具**
   - 导出后使用在线压缩工具
   - 例如：TinyPNG、ImageOptim
   - 通常可以减小 30-50%

4. **分批导出**
   - 导出 4-6 张为一个文件
   - 然后在其他工具中合并

---

#### Q15: 导出的分镜编号不对

**A:** 分镜编号基于选择顺序。解决方案：

1. **按顺序选择分镜**
   - 从左到右、从上到下选择
   - 或使用 Ctrl+A 全选

2. **使用 Shift+点击 逐个选择**
   - 这样可以控制选择顺序
   - 编号会按选择顺序生成

3. **重新排列分镜**
   - 在画布上拖拽分镜
   - 调整到正确的位置
   - 然后重新导出

### 第六部分：快速分镜问题

#### Q16: 快速分镜和普通生成有什么区别？

**A:** 快速分镜是预设的生成模板，提供快速生成特定类型分镜的方式。

**对比：**

| 功能 | 普通生成 | 快速分镜 |
|------|---------|---------|
| 灵活性 | 高 | 中 |
| 速度 | 中 | 快 |
| 学习成本 | 高 | 低 |
| 自定义 | 完全自定义 | 预设模板 |

**快速分镜的 4 种类型：**

1. **三视图**：正面、侧面、顶面
2. **多网格**：多个摄像机角度
3. **风格对比**：5 种不同风格
4. **叙事进展**：连续故事场景

**何时使用快速分镜：**
- ✅ 需要快速生成
- ✅ 不确定如何写提示词
- ✅ 需要特定类型的分镜
- ✅ 想要标准化的输出

---

#### Q17: 如何自定义快速分镜模板？

**A:** 修改快速分镜模板的步骤：

1. 点击 **✨ 高级功能**
2. 切换到 **⚙️ 快速分镜** 标签页
3. 点击要修改的快速分镜（如"多网格"）
4. 点击 **⚙️ 设置** 按钮
5. 编辑提示词模板
6. 点击 **保存**

**模板变量：**
- `{frameCount}` - 帧数
- `{subject}` - 主体描述
- `{style}` - 风格
- `{angleDescriptions}` - 摄像机角度描述

**示例：**
```
原始模板：
生成 {frameCount} 个 {subject} 的不同摄像机角度

修改后：
从 {frameCount} 个不同角度展示 {subject}，
使用 {style} 风格，
摄像机角度包括：{angleDescriptions}
```

**重置模板：**
- 点击 **🔄 重置**
- 确认重置
- 模板恢复为默认值

### 第七部分：账户和配额问题

#### Q18: 如何查看我的配额使用情况？

**A:** 查看配额的方法：

1. 点击左侧 **⚙️ 设置**
2. 找到 **配额信息** 部分
3. 显示：
   - 总配额
   - 已用配额
   - 剩余配额
   - 使用百分比

**配额说明：**
- 每个 AI 服务商的配额独立计算
- 配额通常每月重置
- 某些服务商支持购买额外配额

---

#### Q19: 配额用完了怎么办？

**A:** 配额用完后的解决方案：

1. **购买更多配额**
   - 访问 AI 服务商官网
   - 选择付费计划
   - 购买额外配额

2. **等待配额重置**
   - 大多数服务商每月重置一次
   - 查看官网了解重置时间

3. **切换到其他服务商**
   - 使用其他 AI 服务商的配额
   - 例如：从 Gemini 切换到 OpenAI

4. **优化使用**
   - 减少生成次数
   - 使用更简洁的提示词
   - 避免重复生成

---

#### Q20: 为什么我的配额突然减少了？

**A:** 配额减少的原因：

1. **正常使用**
   - 每次生成都会消耗配额
   - 生成越多，配额消耗越快

2. **高成本操作**
   - 视频生成消耗配额最多
   - 高清模式消耗更多
   - 长视频消耗更多

3. **重复生成**
   - 多次生成同一内容
   - 每次都会消耗配额

4. **其他应用使用**
   - 如果 API Key 被其他应用使用
   - 配额会被共享消耗

**节省配额的方法：**
- 减少生成次数
- 避免重复生成
- 使用标准清晰度而不是高清
- 生成较短的视频

---

### 第八部分：技术问题

#### Q21: 应用加载很慢

**A:** 加快应用加载速度的方法：

1. **清除浏览器缓存**
   - 按 Ctrl+Shift+Delete
   - 选择"所有时间"
   - 清除缓存和 Cookie

2. **关闭浏览器扩展**
   - 某些扩展会减慢加载速度
   - 尝试在隐私模式下打开

3. **更换浏览器**
   - 尝试 Chrome、Firefox、Edge
   - 某些浏览器性能更好

4. **检查网络连接**
   - 确保网络速度足够
   - 尝试连接到其他网络

5. **重启应用**
   - 刷新页面
   - 或重新启动开发服务器

---

#### Q22: 生成过程中页面卡住了

**A:** 页面卡住的解决方案：

1. **等待更长时间**
   - 某些生成需要 5-10 分钟
   - 不要立即关闭页面

2. **检查浏览器控制台**
   - 按 F12 打开开发者工具
   - 查看是否有错误信息
   - 截图错误信息

3. **刷新页面**
   - 按 F5 或 Ctrl+R 刷新
   - 重新开始生成

4. **重启应用**
   - 关闭浏览器标签页
   - 重新打开应用

5. **检查网络连接**
   - 确保网络稳定
   - 尝试重新连接 WiFi

---

#### Q23: 浏览器显示"跨域错误"

**A:** 跨域错误通常是网络问题：

1. **检查 API 配置**
   - 确保 API Key 正确
   - 确保 API 端点正确

2. **检查网络连接**
   - 确保能访问 API 服务器
   - 尝试在其他网络上测试

3. **清除浏览器缓存**
   - 按 Ctrl+Shift+Delete
   - 清除所有缓存

4. **尝试不同的浏览器**
   - 某些浏览器对跨域有不同的处理

5. **联系技术支持**
   - 提供错误截图
   - 提供浏览器控制台错误信息

### 第九部分：最佳实践

#### Q24: 如何写出好的提示词？

**A:** 编写高质量提示词的技巧：

**结构化提示词模板：**

```
【场景】描述发生的地点和环境
【主体】描述主要对象或人物
【动作】描述正在发生的动作
【风格】指定艺术风格或视觉风格
【光线】描述光线和阴影
【情感】描述整体氛围和情感
```

**示例：**

```
【场景】现代办公室，落地窗，城市天际线
【主体】一个穿着黑色西装的商人
【动作】坐在办公桌前，认真地看着文件
【风格】电影级，专业摄影
【光线】温暖的阳光透过窗户洒进来，形成对比光影
【情感】严肃、专业、沉思
```

**提示词技巧：**

1. **具体而不是模糊**
   - ❌ "一个人"
   - ✅ "一个 30 岁的亚洲男性，穿着蓝色衣服"

2. **使用形容词**
   - ❌ "一个房间"
   - ✅ "一个宽敞、明亮、现代的办公室"

3. **指定风格**
   - ❌ "一个场景"
   - ✅ "电影级、赛博朋克风格的场景"

4. **描述光线**
   - ❌ "白天"
   - ✅ "金色的夕阳光线，形成戏剧性的阴影"

5. **避免否定**
   - ❌ "不要有人"
   - ✅ "空荡荡的街道"

#### Q25: 如何提高生成效率？

**A:** 提高工作效率的方法：

1. **批量生成**
   - 一次生成多个分镜
   - 而不是逐个生成

2. **使用快速分镜**
   - 对于标准需求，使用快速分镜
   - 节省时间和配额

3. **保存常用提示词**
   - 记录好的提示词
   - 下次直接使用
   - 节省编写时间

4. **使用参考图片**
   - 参考图片可以加快生成
   - 减少需要的文字描述

5. **合理规划**
   - 先生成草稿
   - 然后微调细节
   - 而不是一次性完美
