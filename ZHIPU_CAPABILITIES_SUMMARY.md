# 智谱 AI 功能总结

## 🎯 当前状态

### ✅ 已集成 (可立即使用)

1. **对话生成 (LLM)**
   - 模型: `glm-4`
   - 用途: 生成视频提示词、翻译文本
   - 状态: 完全支持

2. **图片生成**
   - 模型: `cogview-4-250304`
   - 用途: 生成分镜画面
   - 状态: 完全支持

### 📋 已规划 (需要集成)

3. **图片分析 (Vision)**
   - 模型: `GLM-4V`, `GLM-4.5V`, `GLM-4.6V`
   - 用途: 分析参考图片，提取视觉特征
   - 状态: 智谱官方支持，项目中未集成
   - 集成计划: `ZHIPU_VISION_INTEGRATION_PLAN.md`

### ❌ 不支持

4. **视频生成**
   - 原因: 智谱官方暂不提供视频生成 API
   - 替代方案: 使用神马 API 或其他服务

## 🚀 快速开始

### 1. 配置智谱 API
```
设置 → 对话及图像API → 选择"智谱 AI (ChatGLM)" → 粘贴 API Key → 测试连接 → 保存
```

### 2. 使用对话功能
```
创意对话 → 输入提示词 → 生成分镜
```

### 3. 使用图片生成
```
剧本生成 → 输入剧本 → 设置分镜数量 → 开始生成
```

## 🔧 技术细节

### API 端点
```
Base URL: https://open.bigmodel.cn/api/paas/v4
Chat API: /v1/chat/completions
Image API: /v1/images/generations
```

### 支持的模型

#### LLM 模型
- `glm-4` - 通用对话模型
- `glm-4-plus` - 增强版本
- `glm-4-turbo` - 高性能版本

#### 图片生成模型
- `cogview-4-250304` - 最新图片生成模型

#### Vision 模型 (未集成)
- `glm-4v` - 通用视觉理解
- `glm-4.5v` - 高级视觉推理
- `glm-4.6v` - 最新视觉模型
- `glm-4v-flash` - 快速图片分析

## ❓ 常见问题

**Q: 智谱支持视频生成吗？**
A: 不支持。智谱官方暂不提供视频生成 API。建议使用神马 API。

**Q: 智谱支持图片分析吗？**
A: 支持。通过 GLM-4V 系列模型支持，但项目中未集成。可参考 `ZHIPU_VISION_INTEGRATION_PLAN.md` 集成。

**Q: 生成速度如何？**
A: 智谱是目前最快的，通常比其他服务商快 30-50%。

**Q: 成本如何？**
A: 智谱是最便宜的选择，成本约为 OpenAI 的 1/5。

**Q: 生成质量如何？**
A: 质量稳定，对中文内容优化。图片生成质量与神马相当。
