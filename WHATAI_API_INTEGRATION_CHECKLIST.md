# WhatAI API 集成清单

## 📋 清单 1：需要从 WhatAI 获取的资料

### API 基础信息
- [ ] API Base URL（例如：`https://api.whatai.cc`）
- [ ] API 版本号
- [ ] 认证方式（Authorization Header 格式）

### API 端点
- [ ] 创建视频端点（HTTP方法 + 路径）
- [ ] 查询视频状态端点（HTTP方法 + 路径）
- [ ] 获取配额/余额端点（HTTP方法 + 路径）
- [ ] 其他相关端点（如有）

### 请求格式
- [ ] 创建视频的完整请求体示例（JSON格式）
- [ ] 必需参数列表（参数名、类型、说明）
- [ ] 可选参数列表（参数名、类型、默认值、说明）
- [ ] 参数取值范围（视频时长、宽高比、质量等）

### 响应格式
- [ ] 创建视频的成功响应示例（完整JSON）
- [ ] 查询状态的成功响应示例（完整JSON）
- [ ] 获取配额的成功响应示例（完整JSON）
- [ ] 错误响应示例（错误情况下的JSON格式）

### 业务逻辑
- [ ] 视频状态列表（所有可能的状态值）
- [ ] HTTP 错误码及含义（401、403、429、500等）
- [ ] 业务错误码（如有自定义错误码）
- [ ] 轮询建议（建议多久查询一次状态）
- [ ] 超时时间（视频生成的最长等待时间）

### 限制和配额
- [ ] 单次请求限制（最大视频时长、最大图片数等）
- [ ] API 调用频率限制（QPS - 每秒请求数）
- [ ] 配额计算方式（如何计算消耗）
- [ ] 并发请求限制（是否有限制）

### 其他信息
- [ ] 支持的模型列表
- [ ] 支持的视频输出格式（mp4、webm等）
- [ ] 支持的宽高比（16:9、9:16、1:1等）
- [ ] 支持的视频时长（10s、15s、25s等）
- [ ] 是否支持图片输入（用于视频生成）
- [ ] 官方API文档链接

## 🔄 集成流程

1. **获取资料** → 从 WhatAI 获取清单1的所有信息
2. **修改代码** → 根据获取的资料修改 `videoService.ts`
3. **提供信息** → 如需要，提供清单2的信息给 WhatAI
4. **测试集成** → 测试API连接和功能
5. **部署上线** → 部署到生产环境

- 所有资料都应该以官方文档或API提供商提供的信息为准
- 如有不清楚的地方，直接向 WhatAI 技术支持咨询
- 保存好所有的API文档和示例代码，便于后续维护
